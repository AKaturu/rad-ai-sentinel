# dicom-dose-audit

**An open-source CT radiation-dose audit tool: read CTDIvol and DLP from DICOM
metadata or dose structured reports, group studies by protocol / scanner /
patient-size category, detect missing dose metadata, flag statistical
outliers, compare protocol versions, plot monthly trends, and generate
quality-improvement reports.**

> See `docs/` for the full design. (README completed at the end of the build.)
